# Sample Screenshots

Screens to capture for demo submission:

1. Dashboard with total claims, approved, rejected, partial approval, manual review, recent claims, and decision distribution.
2. Submit Claim page showing prescription, medical bill, diagnostic report uploads, and progress.
3. Claim Processing page showing all five completed stages.
4. Claim Details page showing uploaded documents, extracted information, policy evaluation, triggered rules, fraud signals, confidence score, and decision explanation.
5. Manual Review Queue showing flagged claims, review reason, confidence, fraud signals, and override actions.

Recommended commands after running the frontend:

```bash
npm run dev
```

Open `http://localhost:3000` and capture the five routes above.
